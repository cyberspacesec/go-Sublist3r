Go语言实现的子域名枚举工具

go-Sublist3r 是一个用Go语言实现的Sublist3r子域名枚举工具包装器。 它提供了一种使用Docker运行Sublist3r的便捷方法，无需在系统上直接安装Python依赖项。

特性

Docker集成

自动使用Docker Hub上的公共镜像或构建Sublist3r的Docker镜像，无需手动配置环境

一致的命令行界面

与原始Sublist3r选项一致的命令行界面，让熟悉原工具的用户可以无缝切换

自动下载

首次运行时自动下载或创建Docker镜像，简化用户的初始配置过程

文件处理

自动处理容器和主机之间的输出文件传输，使用户可以方便地操作扫描结果

降级模拟

当Docker不可用时提供模拟功能，确保工具在各种环境中都能工作

RESTful API

提供RESTful API，支持远程调用和与其他系统集成，更适合自动化工作流

API密钥认证

支持API密钥认证，保护API访问安全，防止未经授权的使用

同步/异步API

提供异步和同步API接口，满足不同的集成需求和使用场景

安装

前提条件

  • Go 1.16或更高版本
  • Docker (可选，如果不可用会使用模拟模式)

从源代码构建

git clone https://github.com/cyberspacesec/go-Sublist3r.git
cd go-Sublist3r
make build

全局安装

make install

使用方法

获取Docker镜像

程序会自动获取镜像，但您也可以使用以下命令显式操作:

从Docker Hub拉取镜像 (推荐)

go-sublist3r pull-docker-image

构建本地Docker镜像

go-sublist3r build-docker-image

运行子域名枚举

基本用法:

go-sublist3r scan -d example.com

使用其他选项:

go-sublist3r scan -d example.com -b -p 80,443 -v -o results.txt

启动API服务器

启动API服务器，允许通过HTTP请求进行子域名枚举:

go-sublist3r api --port 8080 --workers 5 --capacity 100

启动带有API密钥认证的API服务器:

go-sublist3r api --port 8080 --api-key "your-secret-key"

API使用

go-Sublist3r提供了RESTful API，支持远程调用和与其他系统集成。

API端点

POST /api/v1/scan

启动一个新的子域名枚举扫描任务

POST /api/v1/scan/sync

启动一个同步扫描并等待结果（最多5分钟）

GET /api/v1/scan/{id}

获取指定扫描任务的状态和结果

GET /api/v1/scans

获取所有扫描任务的列表
